# Dashboard Perusahaan

Project Next.js + Tailwind CSS untuk dashboard TV monitoring DSN Group dengan data dummy.

## Menjalankan Project

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Struktur Data Dummy

Angka dashboard berada di `src/lib/dashboard-data.ts`, sehingga mudah diganti menjadi data API nanti.
